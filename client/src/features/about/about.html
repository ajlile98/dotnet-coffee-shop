About Coffee Shop Demo

Where passion for coffee meets modern web development

The Project

Welcome to my Coffee Shop Demo - a full-stack web application born from two of my favorite things: great coffee and clean code! This project serves as my playground for exploring modern web technologies, specifically diving deep into C# and .NET ecosystem.

Learning Goals

  • Backend Mastery: C# and .NET Core API development
  • Database Skills: Entity Framework Core and data modeling
  • API Design: RESTful services with proper authentication
  • Frontend Bonus: Angular and TypeScript development
  • Full-Stack Integration: Connecting all the pieces together

Tech Stack

Backend
.NET Core, Entity Framework, JWT Auth
Frontend
Angular, TypeScript, HTML/CSS
Database
SQLite with EF Core migrations
DevOps
Docker, containerized deployment

Current Features

This is very much a work in progress, but here's what's brewing:

  • User registration and authentication system
  • Dynamic menu display with pricing
  • Shopping cart functionality
  • Order creation and submission
  • Backend validation and data persistence
Coming Soon: Menu management, order tracking, employee queue system, and admin controls. Stay tuned!
